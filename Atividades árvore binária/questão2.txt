1. Árvore Rubro-Negra (Red-Black Tree)
Uma árvore rubro-negra é uma estrutura de dados de árvore que ajuda a manter os dados organizados de forma balanceada, permitindo buscas, inserções e deleções rápidas. 
Ela é usada em sistemas de banco de dados e algoritmos onde é importante que o tempo de acesso aos dados seja consistente. 
A árvore segue regras para que a altura dos ramos (do topo até as folhas) se mantenha equilibrada, garantindo que nenhuma parte fique muito mais "pesada" que as outras.
Assim, toda vez que um elemento é adicionado ou removido, ajustes são feitos para manter esse equilíbrio.

2. Árvore B
A árvore B é uma árvore mais "larga" que permite vários valores por nó, diferentemente de uma árvore binária que permite apenas dois filhos. 
Ela é útil em bancos de dados e sistemas de arquivos grandes, especialmente porque sua estrutura permite ler e gravar muitos dados de uma só vez. 
Cada nó pode armazenar um grande número de valores e tem várias "saídas" para os próximos nós, o que torna as buscas e operações eficientes, mesmo em sistemas de armazenamento em disco.

3. Árvore B+
A árvore B+ é uma versão aprimorada da árvore B, onde os dados reais são mantidos apenas nas folhas. 
As folhas ficam ligadas umas às outras, como uma lista encadeada. 
Isso facilita a leitura sequencial dos dados, sendo muito útil em bancos de dados para consultas de intervalos, onde, após encontrar o primeiro item, é fácil ler os dados restantes na sequência.
